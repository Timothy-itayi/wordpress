

# Product Manager

A PHP CRUD application built as a learning exercise. The goal was to understand PHP's core web mechanics, request handling, form processing, database interaction via PDO, and server-side HTML rendering.

The app manages a single entity (products) backed by SQLite. Each PHP file maps directly to one operation: list, create, edit, or delete. Just the raw request/response cycle that every PHP framework is built on top of.



cd product-manager
php -S localhost:8000
